# ⚡ 电力世界 · PowerWorld

一款**以电网本身为主角**的硬核建造经营游戏（MVP 垂直切片）。

你不是简单地放一个电厂、看一个"供电够不够"的数字——你经营的是一张**活的电网**：
潮流按线路阻抗真实分配、线路有热极限会过载跳闸并**连锁停电**、机组无法瞬时响应、
新能源靠天吃饭、频率随供需失衡波动、燃料与碳排放压着现金流。

**Phase 2 已加入**：高压/中压两级电网与**变电站变压器过载保护**、**储能电池**（充放电消纳新能源）、
**天气与危机事件**（热浪/寒潮/无风/阴雨/风暴损线）、**三关战役 + 存档读档**。

**Phase 3 已加入**：**N-1 冗余校核**（逐一停运校验失负荷/过载）、**科技树/研发**（研发点解锁
超高压/高效机组/需求响应/大容量变压器/先进储能/自动重合闸）、**公众形象与减碳**（停电/碳强度/
临近污染影响口碑，口碑反过来影响等效电价；清洁电力占比指标）。

**Phase 4 已加入**：**新手教程**（手把手分步引导）、**沙盒模式**（无限资金、无输赢）、
**成就系统**（10 项成就跨存档持久化）、**合成音效 + 静音开关**（WebAudio 现场合成、无素材文件）。

**经济系统（已做全面）**：
- **建设工期**：下单即付 capex，工期内不投运不发电（核电 8 天最长、燃气 1.5 天最短，风/光居中）。
- **固定运维成本**：投运资产按日计费，抑制过度建设。
- **燃料市场**：煤/气/铀价格指数波动（均值回归+随机游走+偶发跳涨），改变 merit-order 与成本——天然气最敏感、铀最稳。
- **现货电价**：电价随**稀缺（备用率）与燃料**动态定价，峰时/缺供价格尖峰 → **调峰机组与储能套利**有真实收益，过剩时折价 → 储能低价充电。
- **贷款融资**：举债摊平工期现金流压力，信用额度随资产上升，日利率含负债率风险溢价；**净资产 = 现金+资产−负债**。
- **机组老化 / 强迫停运 / 退役**：投运后磨损上升 → 边际成本与运维上浮、强迫停运概率增大；退役按役龄折旧返还**残值** → "何时换新"的生命周期决策。
- **差异化电价**：居民/商业/工业按类计费（居民溢价、工业批发折扣）→ 服务对象的收入结构成为权衡。
- **碳配额交易（cap-and-trade）**：按电量免费发放配额、基准随时间收紧；**低于基准卖配额获利、高于则买入付费** → 减碳直接变现。
- **远期套保**：对一部分电量签**差价合约**锁价，平抑现货波动（保险费换确定性）。
- **燃料长约**：按燃料类型锁定价格指数一段时间，对冲燃料涨价。
- **计划检修**：主动大修电厂（短暂离线+费用）降役龄/故障率，对冲"听天由命"的强迫停运。
- **绿证交易**：新能源发电获**绿证补贴**（价格随政策退坡），与碳配额互补——奖清洁、罚污染。
- **设备保险**：缴日保费，**赔付强迫停运与风暴损毁** 80% 成本，对冲意外巨额支出。
- **信用评级**：由净资产/杠杆/可靠性/盈利综合出评级(AAA..D)，**直接决定贷款利率与融资上限**。
- **批发市场互联**：接入联络线后电源不足可**购电补缺**（按市场均价加价、受容量限制），避免停电但成本高。
- **经济周期**：宏观繁荣/衰退周期性振荡，需求随之 ±15% → **投资时机**成为权衡（衰退期布局、繁荣期收割）。
- **📈 走势图**：现货电价 / 净资产 / 系统需求的历史折线，便于看趋势、择时套保与投资。
- **多公司竞价市场**：你与 AI 竞争对手按报价排序对区域需求出清，**市场份额**取决于你是否够便宜——越有竞争力获客越快。
  竞争对手**动态演化**（盈利扩张、亏损退役），市场自平衡、护城河随时间被侵蚀。
- **容量拍卖**：容量价由"区域容量目标 vs 总可用容量"出清——紧张则飙升、过剩则归零，奖励适时建设备用。
- **碳捕集(CCS)改造**：给火电加装碳捕集，捕碳 90% 但边际成本上浮——现有火电的减碳路径。
- **输电阻塞节点价差**：线路接近热极限即产生**阻塞成本**，奖励就近发电与扩建输电。
- **长时储能**：电池(4h)/抽水蓄能(12h)/氢储能(60h 多日)——跨多日套利与消纳新能源，长时储能容量信用更高。
- **需求响应**：高价时段付激励让用户**自愿削峰**，平抑尖峰、替代昂贵调峰/购电。
- **跨区外送**：被弃的过剩(清洁)电量**卖入批发市场**，尤其在区域稀缺时获利，奖励超建廉价清洁电源。
- **碳关税**：进口电力承担**碳边境调节税**（进口越多、碳价越高越贵），推动清洁自发自用。
- **电力期权**：付权利金买**看跌(保底)/看涨(封顶)**，单向保护，保留有利方向的收益（与对称的远期互补）。
- **ESG 评级**：清洁/可靠/治理综合评分，高 ESG 享**绿色融资利率折扣**——可持续经营降低融资成本。
- **容量市场**：按**可用确定性容量**获容量补偿，奖励保留备用 → 让调峰/备用机组得以生存（新能源容量信用低）。
- **💹 投资对比面板**：按当前电价/燃料实时算各机组度电成本/日均毛利/回本周期。
- **📊 财务报表面板**：资产负债 + 每日损益（售电分类/燃料/碳/运维/利息/罚款/套保/绿证/保险/净）+ 市场行情 + 碳·绿色市场 + 燃料长约/套保/保险/贷款操作。

> 核心玩法循环：**调度发电 → 直流潮流求解 → 频率/拥堵 → 跳闸保护 → 经济结算**

## 怎么玩

进入游戏先在**主菜单选择关卡**（或「继续上次存档」）。新手建议先玩 **＋ 新手教程**；想自由实验可选 **★ 沙盒**。关卡开始处于**暂停**状态，先布网再开跑：

1. 强制拓扑：**电厂 →[高压]→ 变电站 →[中压]→ 负荷**。所以要先建**变电站**做枢纽，再拉线。
2. 备足电源（燃煤=便宜基荷但爬坡慢、燃气=灵活调峰但贵、风/光=零燃料但间歇、核电=巨大基荷），
   必要时加**储能**填爬坡缺口、消纳新能源、平滑峰谷。
3. 点 **▶**（或空格）开始推进时间。
4. 盯住顶栏：**频率**偏离 50Hz 说明供需失衡；线路变**红**接近热极限、持续过载会**跳闸**；
   **变电站**通过量超额定会**变压器跳闸**切断下游；留意**天气**栏的热浪/风暴等冲击。
5. 目标随关卡而定（如撑到第 12 天且可靠性≥92%），同时别破产。

**操作**：鼠标拖拽平移 · 滚轮缩放 · 数字键 1-9 切换工具 · 空格暂停/继续 · Esc 取消连线 ·
用「检查/重合闸」点击跳闸的**线路或变电站**可恢复送电。

顶栏右侧：**🔬 研发** · **🏆 成就** · **💹 投资对比** · **📊 财务报表/贷款** ·
**N-1**（冗余校核） · **🔊 音效** · **💾 存档** · **☰ 菜单/关卡**。
新建的电厂/变电站/储能/线路会先**在建中**（半透明+黄环+剩余工期），工期结束才投运发电。
顶栏指标含资金、**净资产**、频率、发电/需求、**现货电价**、线损、可靠性、碳排、清洁占比、口碑、天气、研发点。

这让我们能单测、跑无头平衡测试、未来替换渲染器或接入自动调度 AI。

## 硬核仿真要点

- **直流潮流**：`P = B·θ`，潮流按 `1/X` 分配，真实产生拥堵与连锁跳闸（见 `powerflow.test.ts`）。
- **经济调度**：可调机组按边际成本 merit-order 出力，受**爬坡率**约束 → "机组无法瞬时响应"的张力。
- **频率**：由供需失衡推算，过低触发甩负荷（停电）。
- **孤岛**：线路跳闸后电网分裂，无电源的孤岛整片停电。
- **线损**：`loss ≈ R·I²`，远距离输电要付出代价（这也是要升压输电的动机）。
- **经济**：上网电价收入 − 燃料 − 碳成本 − 失负荷罚款；碳价逐日上涨制造减碳压力。
